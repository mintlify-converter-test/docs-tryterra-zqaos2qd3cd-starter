**Instance method of TerraRT**

```swift
func connectDevice(_ device: Device, _ connectionCallback: @escaping(Bool) -> Void)

```

*   `device`: Device -> A device object returned from \`startBluetoothScan
*   `connectionCallback`: @escaping(Bool) -> Void -> A completion function called when the connection is completed.
