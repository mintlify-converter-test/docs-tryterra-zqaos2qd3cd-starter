# FSLSensorDetails

```kotlin
data class FSLSensorDetails(
    val sensor_state: String,
    val status: String,
    var serial_number: String,
    var data: TerraGlucoseData,
    var device_data: TerraDeviceData,
}

```

`sensor_state` can be one of the following:

```
NOT_STARTED
STARTING
READY
FAILURE
SHUTDOWN

```
